---

## 🎮 How to Play

1. **Setup:** Enter your team names in the initial prompt.
2. **Roles:** Use the **Assign Random Roles** button to determine the first Chaser and Evader.
3. **The Chase:** - A standard match consists of **16 rounds**.
   - Each round has a **20-second timer**.
   - **Scoring:** Only the Evader scores a point by surviving the full 20 seconds or clicking 'Evade'.
   - **Tagging:** If a Chaser tags the Evader, no points are awarded, and roles swap for the next round.
4. **Sudden Death:** If the match ends in a tie, the system automatically triggers a Sudden Death chase to determine the ultimate winner.

## ⌨️ Desktop Shortcuts
If running on a laptop or desktop, you can use these keys for faster officiating:

| Key | Action |
| :--- | :--- |
| **Space** | Start Timer / Assign Roles |
| **T** | Tag (Chaser Wins / Swap Roles) |
| **E** | Evade (Evader Wins / +1 Point) |

## 📱 Mobile Optimization
For the best experience on iOS or Android:
1. Open the link in your mobile browser.
2. Select **"Add to Home Screen"** from your browser options.
3. This allows the app to run in full-screen mode without browser address bars interfering with the controls.
